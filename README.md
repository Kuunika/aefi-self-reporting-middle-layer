## Environment Variables
Copy the contents of .sample.env to an new .env file. the variables:

AEFI_OHSP_USERNAME,
AEFI_OHSP_PASSWORD,
AEFI_OHSP_URL

will need to be set before the application can be run.


## Installation and Execution
```bash
npm install --only=prod
npm run start:prod
```
